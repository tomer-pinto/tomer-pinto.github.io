Projects

Weather Data Analysis

A Python project developed in Google Colab that collects and analyzes weather data from 1,000 cities using the OpenWeatherMap API. After cleaning the data, key metrics were explored through visualizations to uncover patterns, trends, and climate extremes.

Academic Management System

A multi-layered academic management system built in Python, applying OOP and MySQL. Supports role-based access for manager, teachers, students, parents, and general workers with modules for courses, enrollments, payments, and maintenance tracking.

Stroke Detection Dashboard

An end-to-end deep learning project for preliminary stroke detection, resulting in an interactive Streamlit dashboard. Trained and evaluated multiple CNNs on a challenging, imbalanced public dataset, creating a final Ensemble model with 88% accuracy.
